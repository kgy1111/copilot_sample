# GitHubスマホアプリにおけるAIモデル選択とプレミアムリクエスト

## 概要

GitHubスマホアプリ（GitHub Mobile）の Copilot Chat では、AIモデルを選択して利用できます。
選択するモデルや契約プランによって、プレミアムリクエストが発生するかどうかが変わります。

---

## プレミアムリクエストとは

プレミアムリクエストとは、高度なAIモデルや特定のCopilot機能を使用した際に消費されるリクエスト枠のことです。
毎月のプレミアムリクエスト枠はプランごとに異なり、月初（UTC 0時）にリセットされます。

---

## スマホアプリでのモデル選択とプレミアムリクエストの関係

### 有料プラン（Copilot Pro / Business / Enterprise）

| 使用モデル | プレミアムリクエスト消費 |
|---|---|
| 含まれるモデル（GPT-5 mini / GPT-4.1 / GPT-4o） | **消費しない** |
| プレミアムモデル（Claude Sonnet 4など） | **消費する**（モデルの倍率に応じて） |

- 有料プランでは、標準で含まれるモデル（GPT-5 mini・GPT-4.1・GPT-4o）を使用してもプレミアムリクエストは消費されません。
- Claude Sonnet 4（1× 倍率）など、プレミアムモデルを選択するとプレミアムリクエストが消費されます。
- プレミアムリクエストを使い切ると、自動的に無料モデルにフォールバックします。

### Copilot Free（無料プラン）

- **すべてのチャット操作がプレミアムリクエストとしてカウント**されます。
- 月50件のプレミアムリクエストが含まれます。上限に達すると利用できなくなります。

---

## iOS / Android のアプリ内購入で加入した場合の注意事項

> **重要**: iOSまたはAndroidのアプリ内購入でCopilot ProまたはCopilot Pro+に加入した場合、**追加プレミアムリクエストを有効にすることができません**。

追加リクエストが必要な場合は、以下の手順が必要です：
1. モバイルサブスクリプションをキャンセルする
2. Webブラウザから GitHub.com で改めてサブスクリプションに加入する

詳細はGitHubサポートポータル（https://support.github.com）にお問い合わせください。

## モデル倍率（参考）

| モデル | 倍率 | 備考 |
|---|---|---|
| GPT-5 mini | 含まれる | 有料プランでは消費なし |
| GPT-4.1 | 含まれる | 有料プランでは消費なし |
| GPT-4o | 含まれる | 有料プランでは消費なし |
| Claude Sonnet 4 | 1× | プレミアムリクエスト消費 |
| Claude Opus 4.5 | 3× | 1回のやり取りで3件消費 |

> ※ モデルの倍率および含まれるモデルは変更される場合があります。
> 最新情報は [GitHub公式ドキュメント](https://docs.github.com/en/copilot/managing-copilot/monitoring-usage-and-entitlements/about-premium-requests) を参照してください。

## まとめ

- スマホアプリでのAIモデル選択は、**選択したモデルによってプレミアムリクエストが発生します**。
- 有料プランで含まれるモデル（GPT-5 mini / GPT-4.1 / GPT-4o）を使えば、プレミアムリクエストは消費されません。
- プレミアムモデル（Claude Sonnetなど）を選択すると消費されます。
- アプリ内購入での加入者は追加リクエストが購入できないため注意が必要です。
